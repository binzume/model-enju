#### pmdの生成メモ

MMDモデル
----------

生成方法メモ．

1. enju.mqoの曲面をフリーズした上で，enju_bone.mqoを追加
2. メタセコイアのプラグインからpmdをエクスポート
3. PMXEditorで開いて*.csvをゴニョゴニョ

自動化するツールもあとで公開するかもですが，まだしっくりくるやり方が見つかってないのでどうなるかわかりません．
